# 🌊 Subsea Engineering Journey v2.0

> **Um mergulho interativo na engenharia submarina através do Three.js e IA Agêntica.**

Este projeto é a segunda evolução do meu site pessoal ([abilioassis.com.br](https://abilioassis.com.br)). Ele combina a precisão técnica da engenharia submarina da **Petrobras** com o poder visual do **React Three Fiber** e a inteligência do **Google Gemini**.

O site foi concebido como um "Gêmeo Digital Educacional", onde o usuário mergulha da superfície até o leito marinho para explorar ativos de produção de petróleo em águas ultraprofundas.

---

## 🛠️ Stack Tecnológica

| Camada | Tecnologia |
| :--- | :--- |
| **Framework Web** | [Astro](https://astro.build/) (Islands Architecture) |
| **Motor 3D** | [React Three Fiber](https://docs.pmnd.rs/react-three-fiber) / [Three.js](https://threejs.org/) |
| **Estilização & UI** | [Tailwind CSS v4](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/) |
| **Estado Global** | [Zustand](https://github.com/pmndrs/zustand) |
| **IA Generativa** | Google Veo (Vídeo), Gemini (Conteúdo/Code), MusicFX (Áudio) |
| **IDE Agêntica** | Google Antigravity |

---

## 🏗️ Metodologia de Desenvolvimento

O projeto é construído através de lições estruturadas, seguindo uma abordagem **Top-Down** e orientada a **User Stories (BDD)**. Cada funcionalidade é documentada em `/docs/user-stories/` antes de ser implementada pelo agente de IA.

### 🗺️ Roadmap de Lições:
- [x] **Lição 1: Fundação** (Setup Astro + R3F + Fallback Strategy)
- [x] **Lição 2: Atmosfera Abissal** (Shaders de Fog, Caustics e Áudio Espacial)
- [ ] **Lição 3: Ativos de Engenharia** (Modelos PBR de ANM e Manifolds com Draco)
- [ ] **Lição 4: Scrollytelling** (Navegação guiada da superfície ao fundo)
- [ ] **Lição 5: Operação de ROV** (Física com Rapier e Interatividade)
- [ ] **Lição 6: HUD & Gemini** (Interface de comando e IA especialista)
- [ ] **Lição 7: Polimento** (Post-processing e Integração com Blog Astro)

## 🌊 A Experiência (Status Atual)

Atualmente, o projeto conta com:
- **Ambiente Imersivo:** Névoa exponencial calibrada para visibilidade técnica.
- **Iluminação Dinâmica:** Sistema de luzes de ROV com projeção de *Caustics* animados.

- **Design Sonoro:** Loop de áudio abissal de baixa frequência para imersão sensorial.

## 🚀 Como Rodar o Projeto

Este projeto roda sobre o **WSL (Windows Subsystem for Linux)**.

1.  **Clonar o repositório:**
    ```bash
    git clone https://github.com/SEU_USUARIO/subsea-journey-v2.git
    cd subsea-journey-v2
    ```

2.  **Instalar dependências:**
    ```bash
    npm install
    ```

3.  **Iniciar o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Acessar no navegador:** `http://localhost:4321`
